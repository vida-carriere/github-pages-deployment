GitHub and Bash Commands



Git Commands Action
git init Initialize a new Git repository.
git clone (repo_url) Clone an existing repository from GitHub.
git add (file) Stage Changes to be committed.
git add. Stage all changes in the current directory.
git commit -m (message) Commit staged changes with a message.
git push origin main Push commits to the 'main' branch on the remote repository.


Bash Commands Action
cd (directory) Change to specified directory.
ls List all files, including hidden ones, with details.
touch (file) Create a new empty file.
mdkir (directory) Create a new directory.
